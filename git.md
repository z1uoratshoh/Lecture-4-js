# String  в JS
В JS мы можем писать стринг по три вида `Double Quotes` `Single Quotes` и `Backticks` 

* > Double Quotes
```js
console.log("Hello world")
```
* > Single Quotes
```js
console.log('Hello world')
```
* > Backticks
```js
let i = "world"
console.log(`Hello ${i}`)
```
